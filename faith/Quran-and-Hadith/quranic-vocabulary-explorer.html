Section III

Suggested Maintenance Workflow

Current data flow: the explorer fetches /assets/data/quran-gharib-vocabulary-200.json on page load. Edit that file to enrich entries — no rebuild needed, just refresh the page.

Priority enrichment: add verified root, transliteration, and meaning for each entry using al-Rāghib al-Iṣfahānī's al-Mufradāt, Ibn Qutaybah's Tafsīr Gharīb al-Qurʾān, or the Quranic Arabic Corpus. The data_quality object in each entry records what still needs review.
